Question-2: What is the difference between innerHTML, innerText, and textContent ?

Using inner HTML, we can display the text inside it by writing tags inside it and add different types of styles. Using inner text, we can see what is written inside it and using inner text, we can change what is written inside the element.

Question-3: What is event delegation in the DOM?

Event delegation means telling its parent that we want to work inside its guarantor instead of working inside the element. We will use event delegation. This reduces our code and makes it very easy to capture its child inside a friend through an event. That's why we use event delegation.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a behavior of the DOM that when an event occurs on an element, it starts from that element first, then gradually goes up to its parent, then the next parent, and so on. If we want, we can stop this from happening. All of this is event bubbling.
